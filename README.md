# 🚗 ECU Diagnostic Log Analyzer V2

A Python-based terminal application that simulates an Automotive ECU Diagnostic System.
This project was built to practice Python fundamentals while solving a real-world automotive problem.
---
## 📌 Features

- ✅ Add Fault Codes
- ✅ Show All Fault Codes
- ✅ Search Fault Codes
- ✅ Delete Fault Codes
- ✅ Count Total Fault Codes
- ✅ Show High Severity Faults
- ✅ Show Medium Severity Faults
- ✅ Show Low Severity Faults
- ✅ Most Common Fault Code
- ✅ Vehicle Fault Statistics
- ✅ Save Fault Logs to File
- ✅ Load Fault Logs from File
- ✅ Clear Fault Logs
- ✅ Update Fault Information
- ✅ Sort Faults by Severity
- ✅ Export Diagnostic Report
- ✅ Demo ECU Scan Simulation
